## Página principal

La página principal debe contener un resumen de lo que puede encontrarse en la totalidad de las páginas web.
- Primer apartado: "Hola, soy Jesús, una creativa entidad..." adjuntando una foto profesional.
- Segundo apartado: "Y yo soy el Ente, quien sostene la realidad de las Altas Tierras..." con un botón redirijiendo a la página de Las Altas Tierras.
- Tercer apartado: Un grid conteniendo los 3 proyectos más importantes del portafolio y un botón que redirija a todos los proyectos.
- Cuarto apartado: Un resumen de mi biografía y mis intereses, junto a un botón para descargar mi currículum vitae
- Quinto apartado: Botones que dirigen a Linkedin, GitHub e Itch.io, al igual que un formulario para enviar un mensaje al correo electrónico.
